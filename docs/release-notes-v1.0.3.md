## 🚀 升级指南

### 从 v1.0.2 升级

#### 自动升级
```bash
composer update pfinalclub/php-mcp
```

#### 手动升级
1. 更新 composer.json 中的版本号
2. 运行 `composer update`
3. 查看新的文档和示例
4. 参与社区活动

#### 配置更新
- 无需更改现有配置
- 可选使用新的配置选项
- 建议查看部署最佳实践

### 新功能使用

#### 文件操作工具
```php
use PFPMcp\Examples\FileOperations\FileOperations;
```
